:::info

以下 YAML 展示了全局配置文件 (`config/paper-global.yml`) 的结构和默认值。

点击配置项可以查看该项的详细说明。

:::

`paper-global.yml` 文件是 Paper 服务器的全局配置文件，包含了 Paper 独有的更底层的全局配置选项，通常用于高级配置和性能优化。

**请注意：**  `paper-global.yml` 文件中的配置项通常不需要修改，除非您对 Paper 的内部机制有深入了解，并清楚知道修改这些配置项的后果。不当的配置可能会导致服务器运行不稳定或出现严重问题。

该文件位于服务器根目录下的 `config` 文件夹中。

**文件路径:** `config/paper-global.yml`
