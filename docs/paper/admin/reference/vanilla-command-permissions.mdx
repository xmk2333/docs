# 原版命令权限节点

本页列出了 Minecraft 原版命令的权限节点。您可以使用这些权限节点在权限插件中配置玩家对原版命令的访问权限。

**权限节点格式:**

原版命令的权限节点通常以 `minecraft.command.` 开头，后跟命令名称。例如，`/gamemode` 命令的权限节点为 `minecraft.command.gamemode`。

**默认权限等级说明:**

*   **0**:  所有玩家 (包括未授权玩家)
*   **1**:  服务器管理员 (Operator Level 1)
*   **2**:  服务器管理员 (Operator Level 2)
*   **3**:  服务器管理员 (Operator Level 3)
*   **4**:  服务器管理员 (Operator Level 4) - 最高权限等级，通常只有服务器控制台拥有此权限

**如何使用权限节点:**

您可以在权限插件 (例如 LuckPerms, PermissionsEx) 中使用这些权限节点来配置玩家的命令权限。例如，要允许玩家使用 `/gamemode` 命令，您可以在权限插件中为玩家或用户组添加 `minecraft.command.gamemode` 权限。

**示例 (LuckPerms):**

```
/lp user <玩家名> permission set minecraft.command.gamemode true
```

**注意事项:**

*   权限节点区分大小写，请确保输入正确的权限节点名称。
*   部分插件可能使用不同的权限系统或权限节点格式，请参考您使用的权限插件的文档。
*   过度开放命令权限可能会导致安全风险，请谨慎配置命令权限。
